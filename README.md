# Crestpoint Outsourcing 📊

> Official corporate website for Crestpoint Outsourcing, an accounting and financial services firm.

🌎 **[View Live Demo](https://karanbe17.github.io/crestpoint-outsourcing/)**

## 📖 Project Overview
This project is a fully responsive, multi-page corporate website designed to establish a strong digital presence for a financial outsourcing client. The focus of this build was to deliver a highly professional, trustworthy user interface while maintaining peak performance, clean code architecture, and accessibility.

## ✨ Key Features
* **Bespoke UI/UX Design:** Implemented a custom design system using deep slate and vibrant emerald/sapphire accents to balance corporate professionalism with modern web aesthetics.
* **Fully Responsive Layouts:** Fluid layouts built with CSS Grid and Flexbox to ensure a seamless experience across desktop, tablet, and mobile devices.
* **Optimized Performance:** Built entirely with vanilla technologies, avoiding bloated external libraries to ensure lightning-fast load times.
* **Semantic HTML5:** Structured for maximum SEO optimization and screen-reader accessibility.
* **Dynamic Client-Side Interactions:** Vanilla JavaScript implementation for scroll effects, mobile navigation toggling, and form interaction.

## 🛠️ Tech Stack
* **Markup & Styling:** HTML5, CSS3 (Custom Properties/Variables)
* **Logic & Interactivity:** Vanilla JavaScript (ES6+)
* **Version Control & Deployment:** Git, GitHub Pages

## 🏗️ Architecture & Design Decisions
To ensure high performance and low technical debt, this project was built without relying on heavy CSS frameworks like Bootstrap or Tailwind. 

Instead, it utilizes a custom-built, lightweight CSS architecture. Global variables (`:root`) are used for color palettes, spacing, and typography to maintain strict design consistency across all pages. This approach demonstrates strong fundamental knowledge of the CSS Box Model, layout mechanics, and DRY (Don't Repeat Yourself) coding principles.
